# Say hello to the material legacy project

Say hello to the material legacy project.

## Usage

``` r
hello_material_legacy(name = "LTER")
```

## Arguments

- name:

  A character string with a name to greet.

## Value

A character string with a friendly greeting.

## Examples

``` r
hello_material_legacy("LTER")
#> [1] "Hello LTER from the Material Legacy project!"
```
